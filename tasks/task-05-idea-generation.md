# Task 05: Idea Generation

## Goal

지식과 생각을 조합해 창의적 아이디어 후보를 `30_ideas`에 생성한다.

## Requirements

- 모든 아이디어는 input material을 가져야 한다.
- 조합 방식인 `method`를 기록한다.
- 아이디어는 자동 생성 가능하지만 채택/폐기는 사용자가 판단한다.

## Subtasks

- [ ] idea generation method taxonomy 정의
- [ ] distant association rule 정의
- [ ] subtraction/remix/homage 방식 정의
- [ ] idea review checklist 작성
- [ ] experiment candidate 기준 정의

## Output

- `30_ideas/*.md`
- idea review notes
